PONG - remade by Urias

// Controls

At the game screen
	Press 'Esc' to go to the pause screen

	Player 1
		Up = W
		Down = S

	Player 2
		Up = Arrow Up
		Down = Arrow Down

At the pause screen
	Press 'R' to reset the highscores
	Press 'Y' to quit
	Press 'N' to return to the game

// Settings

At the main menu press '0', '1', or '2' to choose the number of players
	0 Players = Game plays itself forever
	1 Player = Keep the ball going as long as possible
	2 Players = Beat the snot out of your friend's score
